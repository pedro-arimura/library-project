# Library

> Criar um sistema de biblioteca

## Entidades:

### Livro
- Nome
- Identificador único
- Autor
- Ano
- Publicadora

### Cliente

- Nome
- Identificador
- Lista Livros Emprestados [Livro, Data do Empréstimo]


### Credor

- Nome
- Identificador
- Lista Livros Emprestados [Livro, Data do Empréstimo, Cliente]
